# ML Utilities

| Скрипт | Назначение |
|--------|-----------|
| `config_utils.py` | Общие функции для обучения/оценки моделей (используется `scripts/run_neural_training.py`, `scripts/finetune_qwen_smoltalk.py`).

Связанные сценарии лежат в корне `scripts/` (`run_neural_training.py`, `finetune_qwen_smoltalk.py`, `train_copilot_model.py`). Настройки описаны в [`docs/06-features/ML_DATASET_GENERATOR_GUIDE.md`](../../docs/06-features/ML_DATASET_GENERATOR_GUIDE.md).
