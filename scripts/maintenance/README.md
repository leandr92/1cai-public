# Maintenance Scripts

PowerShell и Python-сценарии для архивирования и очистки ресурсов на Windows-хостах.

| Скрипт | Назначение |
|--------|-----------|
| `cleanup_project.py` | Общая уборка проекта (удаление временных файлов, отчётов). |
| `final_root_cleanup.ps1` | Финальная уборка корня репозитория перед публикацией. |
| `archive_configurations.ps1` | Архивация конфигураций 1С. |
| `cleanup_old_docs.ps1` | Удаление устаревших документов.

Используйте их после экспериментов или перед передачей артефактов. На Windows запускайте `pwsh` от администратора.
