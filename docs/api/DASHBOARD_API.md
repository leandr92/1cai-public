# Dashboard API Reference

**Version:** 1.0  
**Base URL:** `/api/v1/dashboard`

## Endpoints

### Executive Dashboard
```http
GET /api/v1/dashboard/executive
```

### Owner Dashboard
```http
GET /api/v1/dashboard/owner
```

### PM Dashboard
```http
GET /api/v1/dashboard/pm
```

### Developer Dashboard
```http
GET /api/v1/dashboard/developer
```

### Team Lead Dashboard
```http
GET /api/v1/dashboard/team-lead
```

### BA Dashboard
```http
GET /api/v1/dashboard/ba
```
