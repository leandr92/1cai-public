# Code Review Agents

Специализированные агенты для анализа кода, поиска security-проблем, auto-fix и проверки best practices.

| Скрипт | Назначение |
|--------|-----------|
| `ai_reviewer.py` | Основной AI reviewer. |
| `ai_reviewer_secure.py` | Усиленная версия с security-проверками. |
| `auto_fixer.py` | Попытки исправить найденные проблемы автоматически. |
| `best_practices_checker.py` | Проверка best practices и стандартов coding style. |
| `bsl_parser.py` | Разбор кода BSL для более точных проверок. |
| `security_scanner.py` | Анализ уязвимостей. |
| `performance_analyzer.py` | Оценка производительности и рекомендаций.
