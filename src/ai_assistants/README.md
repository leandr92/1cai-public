# AI Assistants (Legacy Interface)

Базовые классы для AI-ассистентов. Используются исторически и в некоторых интеграциях.

| Файл | Назначение |
|------|-----------|
| `base_assistant.py` | Базовый класс ассистента (интерфейс, общие методы).
| `architect_assistant.py` | Пример реализации для архитектурного ассистента.

Новые агенты находятся в [`src/ai/agents/`](../ai/agents/README.md).
